◉ PROJECT_01

Arduino Radar System

A real-time object-detection radar built with Arduino UNO, HC-SR04 ultrasonic sensor, and a servo motor. Visualised in Processing IDE as a scanning radar display with live distance mapping.

Arduino UNOHC-SR04Servo MotorProcessing IDEC
◈ HARDWARE–SOFTWARE ARCHITECTURE
  1. Servo rotates 0–180°, sweeping the ultrasonic sensor across the field of view in real time.
  2. HC-SR04 emits a 40 kHz pulse; echo return time converted to distance via d = (t × 343)/2 m.
  3. Arduino serialises angle + distance data at 9600 baud over USB to the host computer.
  4. Processing IDE parses the serial stream, maps polar coordinates to screen space, renders the sweep with persistence fade.
  5. Objects within 40 cm highlighted red; clear zones remain green — mimicking a real PPI radar display.

◉ PROJECT_02

Smart Lock System

A password-secured electronic door lock using a 4×4 matrix keypad, LCD display feedback, and servo-driven bolt mechanism. Validates credentials and controls physical access with visual and audio feedback.

Arduino UNOMatrix KeypadLCD 16×2Servo MotorEmbedded C
◈ PASSWORD VALIDATION FLOW
  1. Matrix keypad scanned row-by-row via GPIO polling; debounced presses buffered into a 4-char string.
  2. LCD 16×2 gives real-time feedback: "ENTER PASSWORD" → masked asterisks → "ACCESS GRANTED / DENIED".
  3. Password hash compared against stored credential in EEPROM; timing-safe comparison prevents brute-force timing attacks.
  4. On match: servo rotates 90° to UNLOCK; buzzer emits 2 beeps; LCD shows "WELCOME".
  5. After 5 seconds servo returns to LOCK; 3 failed attempts trigger a 30-second lockout with LED indicator.

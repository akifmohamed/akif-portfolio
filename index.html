◉ PROJECT_01
Arduino Radar System
A real-time object-detection radar built with Arduino UNO, HC-SR04 ultrasonic sensor, and a servo motor. Visualised in Processing IDE as a scanning radar display with live distance mapping.
Arduino UNO HC-SR04 Servo Motor Processing IDE C
HARDWARE–SOFTWARE ARCHITECTURE
Servo rotates 0–180°, sweeping the ultrasonic sensor across the field of view in real time.
HC-SR04 emits a 40kHz pulse; echo return time converted to distance via d = (t × 343)/2 m.
Arduino serialises angle + distance data at 9600 baud over USB to the host computer.
Processing IDE parses the serial stream, maps polar coordinates to screen space, and renders the scanning sweep with persistence fade.
Objects within 40 cm are highlighted in red; clear zones remain green — mimicking an actual PPI radar display.

◉ PROJECT_02
Smart Lock System
A password-secured electronic door lock using a 4×4 matrix keypad, LCD display feedback, and servo-driven bolt mechanism. Validates credentials and controls physical access with visual and audio feedback.
Arduino UNO Matrix Keypad LCD 16×2 Servo Motor Embedded C
PASSWORD VALIDATION FLOW
Matrix keypad scanned row-by-row via GPIO polling; debounced key presses buffered into a 4-char password string.
LCD 16×2 provides real-time feedback: "ENTER PASSWORD" → masked asterisk display → "ACCESS GRANTED / DENIED".
Password hash compared against stored credential in EEPROM; timing-safe comparison prevents brute-force timing attacks.
On match: servo rotates 90° to UNLOCK position; buzzer emits 2 beeps; LCD shows "WELCOME".
After 5 seconds, servo returns to LOCK; 3 failed attempts triggers a 30-second lockout with LED indicator.
